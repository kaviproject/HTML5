Differnece between local storage and session storage?

Session storage

  • It persists data until we close the window
  • Values stored in the session storage is not shared.
  • Maximun size is 5MB

Local storage

  • It persist the data after window or tab closed
  • Values stored shared through all the windows and tabs
  • Maximum size is 10-15MB
